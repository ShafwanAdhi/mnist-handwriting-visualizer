How it Works

layers

Convolutional Layers

The core building block of a CNN. It applies filters to the input to create feature maps that summarize the presence of detected features in the input.

compress

Max Pooling

Reduces the spatial dimensions of the input volume for the next convolutional layer. It helps to reduce computation and controls overfitting.

grain

Dense & Output

After extracting features, the data is flattened and passed to fully connected layers to classify the image into one of the 10 digit classes (0-9).
